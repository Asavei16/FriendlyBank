# FriendlyBank - a friendly app (with JavaScript)
I'm making this app, because I want to learn JavaScript.

You can: 
* Sign In
* Deposit
* Withdraw
* Transfer
* Request loan
* Close account
* Sort balance
* Logout
* Logout Timer
* Currency Conversion
